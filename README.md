# Polynomial Regression (Machine Learning)

This repository demonstrates **Polynomial Regression** using **Python** and **Scikit-learn**.
It is created for learning non-linear relationships in Machine Learning.

---

## What is Polynomial Regression?
Polynomial Regression is a type of regression where input features are transformed into polynomial features to model non-linear relationships.

- Linear vs Polynomial Regression
- Feature transformation
- Model training and prediction
- Model evaluation
